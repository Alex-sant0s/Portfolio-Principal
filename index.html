Courses Taken

HTML

HTML, ou HyperText Markup Language, é a linguagem padrão usada para criar e estruturar páginas na web. Ela utiliza uma série de "tags" para definir elementos como títulos, parágrafos, links, imagens e outros componentes que compõem uma página.

CSS

CSS, ou "Cascading Style Sheets" (Folhas de Estilo em Cascata), é uma linguagem utilizada para descrever a apresentação de documentos HTML ou XML.

JavaScript

JavaScript é uma linguagem de programação de alto nível, amplamente utilizada no desenvolvimento web para adicionar interatividade e dinamismo às páginas.

MySql

MySQL é um sistema de gerenciamento de banco de dados relacional de código aberto, amplamente utilizado para armazenar e gerenciar dados.
